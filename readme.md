## Software laser cutter
Desarrollado e implementado por Miguel Califa 08/06/2020
### Material empleado
* Arduino Mega 2560
* Ramps 1.4
* Driver TB6600
* 2 * Nema 23
* Fuente 12V/20A, 24V/10A
* Laser 12V/ 5 Watt -> Consumia en realidad 20 Watt
* Cable UTP Cat 6 Interperie
